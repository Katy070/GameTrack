# GameTrack

**GameTrack** is a Django-based web application for tracking video games, writing reviews, and managing your gaming collection.

## Features

- Add, edit, and delete games
- Write and display reviews
- Track time played per game (with cumulative logging)
- User registration, login/logout, and profile system with avatars
- Search and filter games by genre, platform, and more
- Dark mode toggle for better UX
- Fully custom CSS styling

## Technologies Used

- Python 3 & Django
- PostgreSQL (or SQLite for development)
- Bootstrap & custom CSS
- Django Template Engine
- JavaScript (for dark mode toggle, etc.)
